

# **Proxipay**

Proxipay is a mobile app that lets people send and receive payments over Bluetooth without needing an internet connection. Two nearby devices can connect, exchange payment data securely, and sync everything later when the network is available.

Built with **React Native**, **Expo**, and **BLE** for fast and reliable offline transfers.

---

## **Features**

* Offline payments using Bluetooth
* Secure device-to-device communication
* Local transaction storage and later sync
* Simple login and account system
* Clean UI for sending and receiving payments
* Works even with limited or no network coverage

---

## **Tech Stack**

* **React Native / Expo**
* **Bluetooth Low Energy (BLE)**
* **Node.js backend (API)**
* Secure local storage for tokens and user data

---

## **Getting Started**

### Clone the repo

```sh
git clone https://github.com/your-username/proxipay.git
cd proxipay/expo-blepay
```

### Install dependencies

```sh
npm install
```

### Start the app

```sh
npx expo start
```

## **Usage**

1. Create an account or log in
2. Enable Bluetooth
3. Discover nearby devices
4. Send or receive payments offline
5. Sync transactions when online

---

## **Why Proxipay?**

People should still be able to pay each other even when the internet is down.
Proxipay focuses on making that simple, fast, and secure.
